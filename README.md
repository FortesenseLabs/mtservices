# Wise Finance Metatrader Interface

Metatrader application interface

## Execution

**For MT5:**

```bash
cd MT5
make run

```

**For MT4:**

```bash
cd MT4
make run

```

Goto http://localhost:8080/

## Notes

- MT4 is not fully supported yet
